# Regional Crop Yield Prediction & Recommendation System

An interactive machine learning application built to predict regional crop yields and recommend optimal crops based on environmental, soil, and agricultural data.

## 🚀 Features
* **Yield Prediction:** Uses advanced machine learning regression to estimate crop production accurately.
* **Crop Recommendation:** Recommends the most suitable crops to grow based on regional input parameters.
* **Interactive UI:** Built with an easy-to-use interface for quick data input and instant visualizations.

## 📊 Machine Learning Models Used
* **Random Forest Regressor** (Primary model for high-accuracy yield estimation)
* **Decision Tree Regressor**

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Scikit-Learn, Pandas, NumPy, Matplotlib, Seaborn
* **Framework:** Streamlit (for the web interface)
